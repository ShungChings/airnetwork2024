Tính Năng Trade Đồ

Giới Thiệu

Hệ thống Trade đồ trên AirNetwork cho phép người chơi trao đổi vật phẩm một cách an toàn và minh bạch, tăng cường trải nghiệm chơi game.

Lệnh Trade

Command Description Permission
/axtrade help List commands axtrade.trade
/axtrade <player> Send a trade request axtrade.trade
/axtrade accept <player> Accept a trade request axtrade.trade
/axtrade deny <player> Deny a trade request axtrade.trade
/axtrade reload Reload plugin axtrade.admin
/axtrade force <player> Force a trade axtrade.admin
